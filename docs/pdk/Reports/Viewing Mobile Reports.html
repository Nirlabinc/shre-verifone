vmobilereportcategorylist vmobilereportcategorylist View the list of available report categories. Currently, there are only three categories available:
■ Terminal Batch Detail - This report includes transaction data from all the MPPAs.
■ Report by Host - This report includes information about one specific Mobile Adapter.
■ Above Site Loyalty Report - View all above site loyalty, above site are loyalty are discounts provided by the Mobile Host, it may or may not be tied to a Loyalty Host, Mobile hosts can provide discount based on digital offers or other discount mechanisms.
vmobilehostlist vmobilehostlist View list of Adapters and Mobile Hosts currently configured and by program name. The program name is manually configured at the site. It is currently defined as alphanumeric.
vmobilereportlist HostName vmobilereportlist View list of reports available for the host provided.
vmobilereport HostName, ReportName, FileName vmobilereport View data for one Mobile Host. Only transactions processed by the mobile host selected will be retrieved.
