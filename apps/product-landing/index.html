Production controls

Pilot Flow

1. Install

Run preflight, install dependencies, protect runtime, and start the local dashboard.

2. Activate

Sign up through Shre Auth, map tenant/workspace/store, and configure Commander credentials.

3. Validate

Ping Commander, pull XML, export CStoreSKU config, and review readiness.

4. Go Live

Enable sidecar, monitor health, use messenger, and backfill queue/usage when connectivity returns.
